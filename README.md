# Hangang Temperature API 🎈

## 1. Explanation

#### 1.1 Function

  - **한강 수온 정보** 상류층 (구리) 한강 수온 정보를 불러옵니다.

## 2. API specifications
#### 1.1 한강 수온 정보

- 한강 수온 정보를 가져옵니다. (상류층)
 
## 3. Response Code
##### Success Response

| http status code | code | message | comment |
| ---------------- | ---- | ------- | ------- |
| 200 | 200 | SUCCESS | 정상 응답 |

##### Error Response

| http status code | code |  comment |
| ---------------- | ---- |  ------- |
| 200 | 204 | NO_CONTENT | 반환되는 정보가 없음 |
| 200 | 408 | 시간 초과  |
